# Financial Formulas

Common financial formulas used in analysis and valuation.

## Basic Formulas

### Return Calculations
- **Simple Return**: $R = \frac{P_t - P_{t-1}}{P_{t-1}}$
- **Log Return**: $r = \ln(\frac{P_t}{P_{t-1}})$
- **Total Return**: Includes price appreciation + dividends

### Time Value of Money
- **Future Value**: $FV = PV \times (1 + r)^n$
- **Present Value**: $PV = \frac{FV}{(1 + r)^n}$
- **Net Present Value (NPV)**: $\sum \frac{CF_t}{(1 + r)^t}$

## Valuation Formulas

### Dividend Discount Model
- **Gordon Growth Model**: $P = \frac{D_1}{r - g}$

### Relative Valuation
- **P/E Ratio**: $\frac{\text{Price per Share}}{\text{Earnings per Share}}$
- **P/B Ratio**: $\frac{\text{Market Cap}}{\text{Book Value}}$
- **EV/EBITDA**: $\frac{\text{Enterprise Value}}{\text{EBITDA}}$

## Risk Metrics

### Volatility
- **Standard Deviation**: $\sigma = \sqrt{\frac{\sum(x_i - \mu)^2}{n}}$
- **Beta**: $\beta = \frac{\text{Cov}(r_i, r_m)}{\text{Var}(r_m)}$

## Portfolio Metrics

### Sharpe Ratio
$$\text{Sharpe} = \frac{R_p - R_f}{\sigma_p}$$

### Information Ratio
$$\text{IR} = \frac{R_p - R_b}{\sigma_{p-b}}$$

## Accounting Formulas

### Profitability
- **Gross Margin**: $\frac{\text{Revenue} - \text{COGS}}{\text{Revenue}}$
- **Operating Margin**: $\frac{\text{Operating Income}}{\text{Revenue}}$
- **Net Margin**: $\frac{\text{Net Income}}{\text{Revenue}}$

### Liquidity
- **Current Ratio**: $\frac{\text{Current Assets}}{\text{Current Liabilities}}$
- **Quick Ratio**: $\frac{\text{Current Assets} - \text{Inventory}}{\text{Current Liabilities}}$
